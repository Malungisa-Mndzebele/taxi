# Product Overview

## Taxi App - Ride Sharing Platform

A full-stack ride-sharing application similar to Uber, enabling passengers to request rides and drivers to accept and fulfill them.

## Core Features

**For Passengers:**
- User registration and JWT authentication
- GPS-enabled ride requests with real-time location
- Live ride tracking with driver location updates
- Real-time fare calculation with surge pricing
- Multiple payment methods (card, cash, wallet)
- Driver ratings and ride history
- Push notifications for ride updates

**For Drivers:**
- Driver registration with vehicle information
- Online/offline status management
- Real-time GPS location tracking
- Accept and manage ride requests
- View earnings and ride history
- Driver statistics and analytics

**Technical Capabilities:**
- Real-time communication via Socket.IO
- Geospatial queries for nearby driver search
- Complete API documentation with Swagger
- Comprehensive test coverage (unit, integration, E2E, performance)
- CI/CD pipeline with GitHub Actions
- Cloud deployment on Fly.io

## Platform Support

- **Mobile**: React Native app for iOS (13.0+) and Android (8.0+)
- **Web**: Progressive Web App (PWA) with React
- **Backend**: RESTful API with real-time WebSocket support
